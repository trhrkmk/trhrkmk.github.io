macOSのodbc.iniの場所

自分用のメモ。

ODBCで設定した情報を 取り除く しても、削除することができなかったので、 odbc.ini を直接編集した。

パスは以下の通り。

/Library/ODBC/odbc.ini

(ins) teruhiro@MacBook-Pro-13:~ $ cd /Library/ODBC/
(ins) teruhiro@MacBook-Pro-13:/Library/ODBC $ cat odbc.ini
[ODBC Data Sources]
name = Actual Open Source Databases

[name]
Driver   = /Library/ODBC/Actual Open Source Databases.bundle/Contents/MacOS/atopnsrc.so
Server   = test.com
Database = database

(ins) teruhiro@MacBook-Pro-13:/Library/ODBC $ cat odbcinst.ini
[ODBC Drivers]
Actual Access                 = Installed
Actual Open Source Databases  = Installed
Actual Oracle                 = Installed
Actual SQL Server             = Installed
MySQL ODBC 8.0 Unicode Driver = Installed
MySQL ODBC 8.0 ANSI Driver    = Installed

[Actual Access]
Driver = /Library/ODBC/Actual Access.bundle/Contents/MacOS/ataccess.so
Setup  =

[Actual Open Source Databases]
Driver     = /Library/ODBC/Actual Open Source Databases.bundle/Contents/MacOS/atopnsrc.so
Setup      =
LicenseKey = hogehoge

[Actual Oracle]
Driver = /Library/ODBC/Actual Oracle.bundle/Contents/MacOS/atoradb.so
Setup  =

[Actual SQL Server]
Driver = /Library/ODBC/Actual SQL Server.bundle/Contents/MacOS/atsqlsrv.so
Setup  =

[MySQL ODBC 8.0 Unicode Driver]
Driver = /usr/local/mysql-connector-odbc-8.0.25-macos11-x86-64bit/lib/libmyodbc8w.so

[MySQL ODBC 8.0 ANSI Driver]
Driver = /usr/local/mysql-connector-odbc-8.0.25-macos11-x86-64bit/lib/libmyodbc8a.so
