Hugoのリンクを新規タブ(target='_blank')で開く設定のメモ

Hugoでブログを書いていて「リンクを新規タブで開くようにしたいなー」と思い、設定したのでメモ。

なにも設定せずにリンク[ぐーぐる](https://google.com)を使うと、以下のようなhtmlになります。

<a href="https://google.com">ぐーぐる</a>

新規タブで開く設定をしたいので、以下のhtmlにしたいです。

<a href="https://google.com" target="_blank">ぐーぐる</a>

config.tomlを編集する

config.tomlにhrefTargetBlank = trueを追加すればOKなはずです。

defaultContentLanguage = "ja"
baseurl = "https://www.trhrkmk.com/"
languageCode = "jp"
title = "HHKBは体の一部"
theme = "themeName"

[blackfriday]
  hrefTargetBlank = true
