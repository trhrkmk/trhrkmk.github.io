Vimをビルドするときのconfigureのメモ(Debian)

Vimをビルドするときのconfigureのオプションをメモっておく。

詳細は vim-jp の記事を見るのが良いと思います。

https://vim-jp.org/docs/build_linux.html

Debianでsudo aptitude install vim-gnomeでインストールした場合terminal(:ter)が使えない…

ということで、ビルドします。

configure

./configure \
  --with-features=huge \
  --enable-multibyte \
  --enable-gui=gnome2 \
  --enable-gpm \
  --enable-cscope \
  --enable-perlinterp \
  --enable-pythoninterp \
  --enable-python3interp \
  --enable-rubyinterp \
  --enable-luainterp \
  --enable-acl \
  --enable-fontset \
  --enable-xim \
  --enable-terminal \
  --enable-fail-if-missing \
  --with-luajit \
  --with-x \

make

sudo make install
